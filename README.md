# US_Emissions
US County GHG Emissions Analysis
This project analyzes greenhouse gas (GHG) emissions at the county and state levels across the United States using Databricks. It includes SQL queries and data processing workflows to explore spatial patterns, identify emission hotspots, and support climate policy research. The repository provides tools for aggregating, visualizing, and comparing emissions data alongside demographic and energy consumption metrics.
